Getting Started

1) Get the Hardware

Subnodes runs off a Raspberry Pi Model B, 8GB class 10 SD card, 2-3xwireless adapater running the rt5370 chipset, and optional battery, if you need to get mobile with your device. The additional wireless adapters can be used for running the mesh point and connecting the Raspberry Pi to the greater internet if data needs to be uploaded or custom apps installed.

2) Download Subnodes

The preconfigured disk-image is not yet available, so you'll have to clone the subnodes repository and run install_subnodes.sh through your Raspberry Pi

3) That's it!

With your Subnodes configured Raspberry Pi, just plug it in anywhere. You'll notice a subnodes wifi appear which when you log into, will let you reconfigure your Raspberry Pi's network settings or application.

About the Project

The Subnodes project is an open source initiative focused on streamlining the process of setting up a Raspberry Pi as a wireless access point for distributing content, media, and shared digital experiences. The device behaves as a web server, creates its own local area network, and does not connect with the Internet. This is key for the sake of offering a space where people can communicate anonymously and freely, as well as maximizing the portability of the networkâ€“no dependability on an internet connection means the device can be taken and remain active anywhere. The devices are also mesh enabled, implementing the BATMAN Advanced routing protocol.

Subnodes networking is set up using linux software packages hostapd and dnsmasq. The latest version of Raspbian comes with the latest BATMAN advanced module installed, so mesh-enabling your device is as simple as turning the module on and running a few commands.

Initial support for Subnodes has been provided by Eyebeam. Source code for the initiating project can be found over on github, which includes an installation script and detailed installation instructions. The version you're seeing added a GUI for headless and easy network reconfiguration and to allow apps to be installed and configured. This version can also be found on github.
